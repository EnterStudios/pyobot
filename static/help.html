Pyobot - help

Pyobot, as a bot supporting various means of communication, is somewhat intimidating at first to use. This Help page is here to help you learn about the basics and see how to operate the notification service.

If, after reading all of this, you still feel you have problems using the bot, feel free to chat to us on Discord and we will be happy to assist you in your quest to catch'em all!

Supported platforms

Currently, the bot operates via two platforms: Facebook Messenger and Discord (both on public channels and via the Direct Messages (DM)).

This help file will apply to both platforms at the same time, unless otherwise stated.

How to start?

After picking the platform you're more comfortable with, the easiest way to start is just sending a simple command to the bot: !register. As you'll notice reading this document, all of the commands will start with exclamation mark. This is done to ease discrimination between bot commands and normal talk.

Tracking and untracking Pokemon

This might be the most important function of the bot - allowing you to select which of the rare Pokemon you want to track.

With the current settings, we allow you to track the following Pokemon:

To set up your tracking, simply use the following command: !track <Pokemon> <distance>. For Pokemon, you can either use its name or ID (so using Dratini and 147 will give you the same result).

Distance can be provided in the following forms:

  • all - will track all the sightings, regardless of where they spawn;
  • 500 - will track the sightings in a given radius around your set location (in metres) - 500 metres in this example;
  • 0.5km - same as above, but specified in kilometres;
  • 0.310686mi - same as above, but specified in miles;

Word of warning here: if you set up the range-based notifications, but do not set up your location (or unset it) - the bot will NOT notify you about those, since it will be unable to calculate the distance between the spawn and your location.

To stop tracking selected Pokemon, just issue: !untrack <Pokemon>. You can also replace Pokemon with all to remove all your set up trackings.

Tracking command examples with explanations

!track Unown all - this command will make you receive a notification about Unown (ID: 201) spawning anywhere in London.

!track Dratini 500 - this command, given you set your location, will notify you about Dratinis spawning within the 500 metres radius of your location.

!track 3 1km - this command, given you set your location, will notify you about Venusaurs (ID: 3) spawning within the 1 km radius of your location.

!track Larvitar 1.5mi - this command, given you set your location, will notify you about Larvitars spawning within the 1.5 miles radius of your location.

Tracking multiple Pokemon

If you want to track more than one Pokemon, just issue individual commands for each Pokemon you want to be tracked.

At the moment, there is no way to do this all in one command, however it is something we are looking into.

Setting and unsetting your location

In order to take advantage of tracking only spawns close to you on the distance basis, you have to set up your location point so that the bot will be able to check if the new spawn is within your range of interest (for nerds: we use Haversine forumla to check).

To set your location, use the following command: !setLocation lat, lng. You can find out the coordinates from Google Maps or services like Where Am I. Please make sure the coordinates you are using are provided in the decimal format, as the bot won't convert other types of coordinates for you. Example: !setLocation 51.501461, -0.141611 - this will set your location to just in front of Buckingham Palace (say Hello to the Queen from us!).

Facebook Messenger users can just share their location directly with the bot via the mobile application - just open the conversation with the bot, click on the Plus symbol located in the bottom left corner and select Location from the menu. Pick the location you want to use as a centre for any distance calculations and notifications and send it to bot. It can't get any simpler, can it?

We will only keep your latest location, which means setting a new one overwrites the previous. However, if you want it completely removed from our database, use: !unsetLocation.

Setting favorite locations

To save time when setting locations, you might want to create a list of your favorite locations to select from.

In order to do so, use command: !saveLocation lat, lng name. Name can be any combination of letters, numbers and dashes.

To use this location, simply replace coordinates in !setLocation command to your set name.

Examples:

!saveLocation 51.501461, -0.141611 BuckinghamPalace will save that as a favorite location. After that, you can do !setLocation BuckinghamPalace at any time to move your tracking centre to that location.

Review your trackings

If you want to check the list of Pokemon you are tracking at any time, simply send the following command: !myTracking and the bot will reply with the full list of your tracking notifications.

Stop and (re)start the notifications

If, at any time, you will feel that you want to stop the notifications, you can always use the following command: !stop. This will prevent the bot from sending you any new information about spawns you are tracking, however, all your settings will still be stored and waiting for your return.

If you want to unpause your notifications, just send: !start and you will start receiving notifications from that point again.

Unregistering

While we are not planning on removing inactive accounts, if you want yours removed from the database, just send following command: !unregister. Be aware that all your custom settings and trackings will be removed and we will not be able to recover them.

I found a bug, what to do now?

First of all - sorry for that. We are striving to provide you with the best experience possible, but unfortunately, as always with the software, errors do happen.

If you believe you found a bug - a command didn't work for you or a notification was missing for example - come to Discord and post about your problem on the #dev channel.
